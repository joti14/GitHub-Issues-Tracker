## 1️⃣ What is the difference between var, let, and const?
Ans: 
'var' is Function-scoped, hoisted and can be re-declared and updated.

'let' is Block-scoped, not hoisted the same way as 'var', can be updated but not re-declared in the same scope. 

'const' is Block-scoped, values can't be reassigned after their initial assignment.

## 2️⃣ What is the spread operator (...)?
Ans: 
The spread operator (...) expands iterables (like an array, string or object) into individual elements. It allows to quickly copy all or part of an existing array into another array or object.

## 3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: 
map() transforms each element in an array and returns a new array with the transformed elements.

filter() creates a new array containing only elements that pass a certain condition and returns the new array.

forEach() executes a function on each element in the array but does not return anything.

## 4️⃣ What is an arrow function?
Ans: 
In JavaScript arrow function is a concise way to write function expressions using the => syntax.
If the function body is a single expression, it automatically returns the value without the return keyword.

## 5️⃣ What are template literals?
Ans: 
Template literals in JavaScript allows multi-line strings, embedded expressions and string interpolation using backticks(``) instead of quotes. Template literals make working with strings more powerful and flexible.
